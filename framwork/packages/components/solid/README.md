# @opentiny/solid

这是一个基于 Solid 框架的组件库：

### 1. 安装

通过运行以下命令下载组件库依赖：

```shell
npm i @opentiny/solid
```

### 2. 引入和使用

在`App.jsx`文件中使用 @opentiny/solid 组件。

```js
import { Button } from '@opentiny/solid'

function App() {
  return (
    <>
     <Button type="danger">Start</Button>
    </>
  )
}

export default App

```
